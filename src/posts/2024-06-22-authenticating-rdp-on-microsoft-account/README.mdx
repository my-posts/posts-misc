RDP(Remote Desktop Connection, 원격 데스크톱 연결)로 연결할 컴퓨터가 Microsoft 계정을 사용하는 경우에는, 사용자 이름에 `MicrosoftAccount\이메일주소`를 입력해야 한다. ~~아마 대부분의 경우 자동으로 입력되어 있을 것이다.~~

비밀번호는 PIN 대신 해당 Microsoft 계정의 비밀번호를 입력해야 된다.

하지만 그 원격 컴퓨터에서 해당 비밀번호로 아직 로그인 한 적이 없다면, RDP로 연결할 수 없다.
