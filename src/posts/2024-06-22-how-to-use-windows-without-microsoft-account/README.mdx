Windows는 인터넷에 연결되면 Microsoft 계정 사용을 강제한다.

처음에 초기화할 때부터, 랜선을 모두 뽑고, 와이파이가 있어도 연결하지 않은 채 진행하면 Microsoft 계정을 사용하지 않고 Windows를 사용할 수 있다.

* * *

언제 막힐지는 모르는데, 지금 먹히는 최선의 방법은 계정 이름에 a, 비밀번호 아무거나 치는 것이다.

지금은 a 계정으로 로그인을 시도하면 로컬 계정으로 안내하는 기능이 숨겨져 있다.
